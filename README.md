# discord.js v14 × discord の OAuth2

## bot

-   言語: TypeScript
-   主に、 discord.js v14 を使用

### bot に置く`.env`ファイルの中身

※ `.env`ファイルに書く内容は、外部に漏れてはいけない内容なので、必ず`.gitignore`ファイルに`.env`を書いてください。

※ 空の文字列(または数字)に適切な情報を入力してください。

```.env
TOKEN = ""
APPLICATIONID = ""
GUILDID = ""
AUTHURL = ""

```
